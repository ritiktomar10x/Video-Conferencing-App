# 🎥 Video Conferencing Web Application

A full-stack real-time video conferencing web application that enables users to communicate through high-quality audio and video calls. The application is built using the **MERN Stack**, **WebRTC**, **Mediasoup**, and **Socket.io**, providing a secure and scalable platform for virtual meetings.

---

## 🚀 Features

* 🔐 User Authentication
* 🎥 Real-time Video & Audio Calling
* 💬 In-Meeting Chat Messaging
* 🖥️ Screen Sharing
* 👥 Create & Join Meeting Rooms
* 🔗 Unique Meeting Links
* 🎤 Mute/Unmute Microphone
* 📹 Enable/Disable Camera

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Socket.io Client
* WebRTC

### Backend

* Node.js
* Express.js
* MongoDB
* Socket.io
* JWT Authentication

---

### Install Dependencies

Frontend

```bash
cd frontend
npm install
```

Backend

```bash
cd backend
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **backend** directory.

```env
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret_key

CLIENT_URL=http://localhost:3000
```

---

## ▶️ Run the Application

Start Backend

```bash
npm run dev
```

Start Frontend

```bash
npm start
```
